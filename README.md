# Jira / Confluence deployment

## Installing recent ansible

On RHEL and CentOS

    sudo yum install ansible

On Ubuntu

    sudo apt-get update
    sudo apt-get install software-properties-common
    sudo apt-add-repository ppa:ansible/ansible
    sudo apt-get update
    sudo apt-get install ansible
